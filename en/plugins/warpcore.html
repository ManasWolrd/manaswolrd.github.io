WarpCore

Spectral Inversion

WarpCore is a multi-band spectral inversion effect plugin inspired by Prosoniq PiWarp and Zynaptiq Wormhole. It splits the spectrum into multiple bands and inverts each band independently, producing sounds ranging from subtle comb filtering to extreme distortion.

Features
  • Multi-band spectral inversion — Split the 0–FreqHigh spectrum into configurable bands and invert each band independently
  • Configurable filter — Adjustable filter pole count and cutoff frequency scaling
  • Formant shifting — Alter timbre without changing pitch
  • Dynamic SIMD dispatch — Automatically utilizes modern CPU SIMD instruction sets
  • Zero-latency processing — Only non-linear phase from Butterworth filters, no added latency
