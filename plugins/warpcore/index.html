WarpCore

频谱反转

WarpCore 是一个多段频谱反转效果器插件，灵感源自 Prosoniq PiWarp 与 Zynaptiq Wormhole。它能将频谱分割为多个频段并对每个频段独立进行频谱反转，产生从微妙梳状滤波到极端扭曲的各种声音质感。

功能特点
  • 多段频谱反转 — 将 0 ~ FreqHigh 的频谱分割为可配置数量的段，对每段分别进行频谱反转
  • 可配置滤波器 — 可调节滤波器极点数量与截止频率缩放
  • 共振峰移动 — 支持共振峰移动（Formant Shifting），改变音色而不改变音高
  • 动态 SIMD 调度 — 自动利用现代 CPU 的 SIMD 指令集进行优化
  • 零延迟处理 — 仅含巴特沃斯滤波器的非线性相位，无额外延迟
